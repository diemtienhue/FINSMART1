# Lệnh Git để Commit và Push

Chạy các lệnh sau trong terminal của VS Code (Git Bash hoặc PowerShell):

```bash
git add components/AdminDashboard.tsx
git commit -m "Fix JSX error: replace -> with → character in AdminDashboard.tsx"
git push
```

Hoặc chạy file batch:
```bash
.\commit-and-push.bat
```
